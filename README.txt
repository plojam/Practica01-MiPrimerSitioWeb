index.html
Esta práctica tiene el tema de deportes, por lo que se creó un index. Html el cual contiene información general sobre los deportes, esta página esta estructurada de la siguiente forma:
Se especifica que usa la estructura de html5, también tiene una etiqueta head donde se especifica que el modo de escritura es utf-8.
El cuerpo de esta página contiene una cabecera, dos secciones (cada sección tiene una etiqueta <u> dentro de un <h1> para resaltar la seccion) y un pie de página; cabecera en la cual se carga una imagen como logo de la página con la etiqueta <img>, también tiene una lista desordenada de navegación entre las diferentes paginas html creadas y también se agrega un bloque donde se encuentra una lista ordenada que permite navegar entre los diferentes artículos que hay en la página.
Una de las secciones contiene 2 articulos, el primero contiene un párrafo donde especifica que es el deporte, y también contiene una etiqueta <aside> para colocar una imagen a un lado del párrafo. El segundo articulo sigue la misma estructura, un párrafo sobre lo que se el deporte profesional y una imagen de apoyo en un lateral de la página, entre una etiqueta <aside>.
La otra sección esta compuesta por 2 artículos los cuales tienen como tema general indagar sobre otros aspectos del deporte. El primer articulo trata sobre el aspecto físico de los deportes, una etiqueta <p> para el texto y una etiqueta <aside> para colocar una imagen de apoyo. El segundo articulo trata de la tecnología implementada en el deporte y esta compuesto por una imagen relacionada al tema y un párrafo donde se especifican los enfoques que te puede tomar la tecnología, resaltando cada enfoque con la etiqueta <strong>.
Para finalizar index.html, el pie de página contiene la información antes requerida (nombre, institución, email, teléfono, copyright), usando (mailto) para el correo electrónico y (tel) para el número de teléfono.

futbol.html
Una de las 8 paginas html que se deben crear es la de futbol.html, la cual fue escogida para tener 4 secciones (cada sección tendrá una etiqueta <u> dentro de un <h1> para resaltar la sección), con 4 artículos cada uno, por lo que este es la pagina mas larga del proyecto. Para empezar la página, se especificó que va a trabajar en html5, también se creo la etiqueta <head> donde se especifica el titulo de la pagina y se indica que el tipo de caracteres que entenderá, el cual es utf-8.
Para el cuerpo, se tiene una cabecera, cuatro secciones y un pie de página; en la cabecera se carga una imagen con la etiqueta <a> como logo de la página, también tiene una lista desordenada de navegación entre las diferentes paginas html creadas y también se agrega un bloque donde se encuentra una lista ordenada la cual nos da la posibilidad de navegar entre todos los artículos de esta página.
La primera sección de esta pagina tiene como tema principal una introducción, esta sección tiene una cabecera donde se le coloca un bloque donde están los enlaces para navegar entre sus propios artículos.
Donde su primer articulo tiene como tema el entender el entender el futbol, por lo que tiene un video descargado y agregado en la carpeta de videos y se lo usa con la etiqueta <video>.
El segundo articulo trata de lo que es el futbol, por lo que se colocó una imagen acompañada de texto relacionado al tema, ubicado al lado de la pantalla con la etiqueta <aside> y dentro de esta etiqueta se encuentra otra la cual es <em> para darle resalte a la palabra al igual para dar un salto de línea se usó la etiqueta <br>, también mediante una lista desordenada donde se nombran minijuegos derivados del futbol.
El tercer artículo de esta sección trata sobre el origen del futbol, por lo que se opto por usar una imagen acompañado de un texto lateral con la etiqueta <aside>.
La segunda sección creada trata sobre reglas, la cual tiene su propio bloque para navegación entre sus artículos; se decidió separar en tres artículos, uno enfocado a reglas de jugadores, otra a reglas de condiciones y la ultima sobre otras reglas. Todos los artículos al tratar sobre reglas, se opto por mantener un mismo formato, el cual consiste en usar una lista desordenada para mencionar las reglas existentes y usando una etiqueta <aside> para colocar a un lateral imágenes de apoyo.
La tercera sección (con su propio bloque de navegacion) trata sobre equipos, por lo cual se dedico un articulo a hablar sobre lo que es un equipo en el futbol dentro de un párrafo, acompañado de una imagen.
El segundo articulo trata sobre las partes que conforman un equipo, por lo que este articulo contiene texto introductorio apoyado con una imagen lateral con la etiqueta <aside>. Aparte, de clasifica la información sobre este tema implementando una tabla con la etiqueta <table> dejando el articulo de una forma mas ordenada.
Para el tercer y ultimo artículo de esta sección, se presentan algunos equipos, por lo cual se usa una etiqueta <p> para el texto, donde también se encuentra una etiqueta <q> para agregar dos palabras entre comillas. Este articulo también contiene una imagen para mostrar equipos de todo el mundo.
En la ultima sección de futbol.html, se incluye un bloque con una lista de navegación para dirigirse directamente a algún articulo de esta sección. Esta sección incluye 3 artículos sonde se muestran los 3 torneos mas importantes del futbol. Por esta razón, los 3 artículos mantienen un mismo formato, el cual consiste en tener una imagen principal acompañado de un texto lateral con la etiqueta <aside>.
Y para finalizar la página, se agrega el pie de pagina general para todas las paginas.
